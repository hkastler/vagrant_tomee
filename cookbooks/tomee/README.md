tomee Cookbook
==============
A cookbook to setup a tomee instance, with a jdbc connection (currently supporting mysql and postgres)


Requirements
------------
VirtualBox
vagrant
berkshelf


Attributes
----------
See the Vagrantfile

Usage
-----
Set your attributes and run vagrant up
